# Blagues-API

Frontend Vanilla JS consommant l'API blablagues.net.

## Fonctionnalités
- Blague aléatoire + réponse cachée
- Copie clipboard  
- Loading state (bouton "Loading..")

## Techno
Vanilla JS, Fetch API, Clipboard API

## Démo
Ouvrir `index.html`

## API Site
[blablagues.net](https://www.blablagues.net/)
